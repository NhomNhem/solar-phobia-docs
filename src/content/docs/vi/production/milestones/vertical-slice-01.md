## Goal

Deliver a vertical slice demonstrating the complete vòng lặp ngày/đêm with:
- phase ban ngày: Soul lựa chọn and ritual assignment
- phase ban đêm: Basic di chuyển and survival mechanics
- Phase transitions with proper camera/audio

## Scope

### Phải Have (MVP)
- Máy trạng thái phase with all 7 trạng thái
- Soul data repository (3 souls: Linh, Van, Minh)
- Basic Day Service UI (lựa chọn only)
- Basic Night di chuyển (WASD, no sprint yet)
- Day→Night transition trigger

### Nên Have (Vertical Slice)
- Full sprint mechanics
- Bộ đếm ward with basic visual feedback
- Simple cover detection
- One curse type (Drag) working

### Nice to Have
- Full tactile rituals hệ thống
- Sensory feedback tiers
- Multiple curse types

## Success Criteria

- [ ] Hoàn tất day/night cycle playable from start to shrine win
- [ ] 2 of 3 souls can be selected and saved
- [ ] phase ban đêm has functional di chuyển and cover
- [ ] Bộ đếm ward counts down and triggers death at 0
- [ ] No S1 bugs in core loop
