## Goal

Deliver a vertical slice demonstrating the complete day/night loop with:
- Day phase: Soul selection and ritual assignment
- Night phase: Basic movement and survival mechanics
- Phase transitions with proper camera/audio

- Phase State Machine with all 7 states
- Soul data repository (3 souls: Linh, Van, Minh)
- Basic Day Service UI (selection only)
- Basic Night movement (WASD, no sprint yet)
- Day→Night transition trigger

### Should Have (Vertical Slice)
- Full sprint mechanics
- Ward Timer with basic visual feedback
- Simple cover detection
- One curse type (Drag) working

### Nice to Have
- Full tactile rituals system
- Sensory feedback tiers
- Multiple curse types

## Success Criteria

- [ ] Complete day/night cycle playable from start to shrine win
- [ ] 2 of 3 souls can be selected and saved
- [ ] Night phase has functional movement and cover
- [ ] Ward Timer counts down and triggers death at 0
